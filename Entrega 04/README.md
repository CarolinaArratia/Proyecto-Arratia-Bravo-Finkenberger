# Entrega 04: Documentación General
## 1. Manual de Identidad Visual
### Referencia al Proceso de Creación y Moodboard
El proceso de diseño de "Especies al Límite" se distanció de las interfaces académicas tradicionales, las cuales suelen ser frías, asépticas y poco atractivas para el público general. Mientras que la selección y balance de la paleta cromática se inspiró en las tonalidades terrosas y vibrantes características de **Owen Davey**, el estilo de ilustración emula la estética de una **bitácora de historia natural** o una guía de campo científica contemporánea.

Este estilo se construye a partir de un entintado de líneas negras firmes y definidas que delimitan los contornos de la fauna, combinado con un trabajo detallado de texturas orgánicas (como el tramado de los plumajes, pelajes y caparazones) que aportan **relieve, tridimensionalidad y realismo.**

El objetivo de este enfoque visual es cruzar la **calidez del territorio** con la **identidad biológica** de las especies locales. Esto se consolidó en el **logo**, una insignia circular que actúa como un ecosistema compacto: entrelaza a tres de los grupos más precarizados y menos carismáticos de nuestra base de datos (un escarabajo, un caracol y un anfibio) coronados por araucarias nativas.

### Justificación de las Secciones Gráficas
**Paleta de Colores y Usos:**

- #FBF8F0 (Fondo - 5%) y #EFEDAO (Recuadros - 5%): Evitan la fatiga visual emulando el papel de una bitácora de campo.
- #1F3D5B (Cuerpo de texto - 25%): Azul profundo que reemplaza al negro puro para suavizar la lectura larga.
- #91532B (Terracota - 15%): Asignado a títulos (H1/H2), menús y crónicas para dar jerarquía y fuerza orgánica.
- #529E47 (Verde - 15%): Diseñado para vincular visualmente los datos y gráficos con el entorno natural.
- #187670 (Teal - 10%) y #86223E (Borgoña - 15%): Tonos de acento para destacar elementos de interfaz e hitos internos.
- #0695D1 (Cian - 10%): Alta visibilidad para botones de acción e hipervínculos activos.

**Tipografías:**

- **Capriola** (80px en títulos): Una tipografía de palo seco (sans-serif) con terminaciones redondeadas y gestos sutiles que recuerdan a la escritura manual. Otorga una personalidad amigable, moderna y de vanguardia a las cabeceras principales. 
- **Lora** (40px en cuerpo): Una tipografía con remates (serif) con curvas contemporáneas, ideal para el formato de crónica periodística de largo aliento. Su diseño facilita la lectura inmersiva y añade el peso literario indispensable para una pieza de no ficción.

**Recursos Gráficos y Botones:**

Se diseñaron botones limpios y sólidos con esquinas redondeadas acompañados de flechas direccionales minimalistas para una navegación sin fricciones.

La ilustración principal del sistema muestra una amalgama densa y superpuesta de biodiversidad chilena. Este recurso visual satura el espacio a propósito, sirviendo como una metáfora exacta de la tesis de la investigación: una fauna que se amontona y lucha por un espacio mínimo para subsistir.
