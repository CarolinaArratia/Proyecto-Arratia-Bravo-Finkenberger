## 4. Documentación - base de datos Wikipedia

### 1. Proceso de Limpieza de Datos
El proceso de limpieza y estructuración de la base de datos digital se realizó a partir de la nómina consolidada de 70 especies del reino Animalia en categorías de amenaza (CR), proveniente de los registros oficiales trabajados por el equipo. La limpieza se ejecutó mediante herramientas de hoja de cálculo, organizando el flujo de trabajo en etapas secuenciales para asegurar que las métricas de tráfico web fueran coherentes con la identidad de cada especie estudiada.

En una primera etapa, se procedió a la Importación y Selección de Variables de Interés Digital. El punto de partida fue la base de datos reducida de 70 especies (Animalia, Endémicas, CR/EN). De este listado, se seleccionaron únicamente las dimensiones necesarias para el rastreo en la web: nombre científico, nombre común y clase taxonómica. El objetivo de esta etapa fue limpiar el exceso de información biográfica para concentrar el esfuerzo en la vinculación digital. Se realizó una búsqueda sistemática en la plataforma de analítica de Wikipedia para cada uno de los registros, asegurando que el artículo consultado correspondiera exactamente al organismo de la muestra. Esta etapa fue crucial para establecer la base sobre la cual se recolectarían las métricas de "curiosidad", evitando la dispersión de datos irrelevantes para el análisis comunicacional.

Posteriormente, se llevó a cabo el Tratamiento de la Invisibilidad Digital. Durante la vinculación, se identificaron numerosos casos donde no existía un registro de visualizaciones o la especie carecía de un artículo propio en la enciclopedia. Siguiendo el criterio de integridad de los registros, se decidió mantener estos casos bajo las etiquetas de "Sin sitio web" o "Sin datos", en lugar de eliminarlos o reemplazarlos por valor cero. Esta decisión es de vital importancia para la calidad del reportaje de no ficción, ya que permite documentar empíricamente qué especies, a pesar de su alto riesgo biológico (como la pancora Aegla denticulata o el caracol Ambrosiella kuscheli), son virtualmente inexistentes para el ecosistema informativo digital. Mantener estos vacíos permite realizar un análisis real sobre la brecha de visibilidad que afecta a la biodiversidad menos "carismática".

En una tercera etapa, se realizó la Estandarización y Limpieza de Métricas Cuantitativas. Las columnas correspondientes al "Total de Visitas" y "Promedio Diario" fueron sometidas a un proceso de limpieza para asegurar su funcionalidad en el análisis estadístico. Se eliminaron ruidos en el formato y se transformaron las celdas a valores numéricos puros, permitiendo así que la base de datos fuera apta para el uso de tablas dinámicas. En los casos donde la herramienta de origen entregaba promedios con múltiples decimales, se optó por una normalización para facilitar la lectura, manteniendo siempre la proporción del tráfico registrado. Se verificó manualmente que los registros con mayor volumen, como el Sapo de Danko (Telmatobius dankoi) con 686 visitas, no presentaran distorsiones por homónimos, validando la autenticidad del interés digital capturado.

Finalmente, se aplicó un criterio de Depuración de Tráfico No Humano. Para garantizar la transparencia y replicabilidad del proceso, se configuró la fuente de datos para excluir las visitas generadas por "agentes" (bots y rastreadores automáticos). Esta decisión técnica es fundamental, ya que asegura que los números consignados en la base de datos representen únicamente la curiosidad de usuarios humanos reales. Sin este proceso de filtrado, las métricas estarían infladas por procesos de indexación de motores de búsqueda, lo que invalidaría cualquier conclusión periodística sobre el interés ciudadano real hacia las especies en peligro en Chile.

### 2. Fuente de datos
Para la construcción de esta base de datos consolidada, se integraron dos fuentes de información:

- **Nómina de especies:** La base de datos se fundamenta en los registros oficiales del Ministerio del Medio Ambiente de Chile, específicamente en el Reglamento de Clasificación de Especies (RCE). Esta fuente fue seleccionada por su carácter oficial y su confiabilidad, proporcionando los datos base sobre taxonomía, categoría de conservación (Peligro Crítico) y endemismo. Estos datos fueron procesados previamente para entregarnos la nómina limpia de 70 especies que sirvió de matriz para nuestra investigación.
- **Wikimedia Pageviews Analysis:** Los datos digitales fueron extraídos de esta herramienta que registra los datos oficiales de la Fundación Wikimedia. Se optó por esta fuente por su transparencia y capacidad de ofrecer métricas objetivas sobre el comportamiento de búsqueda espontáneo. Se utilizó la versión de Wikipedia en inglés (en.wikipedia.org) para capturar una dimensión de impacto global y local, permitiendo analizar cómo se proyecta la crisis de biodiversidad chilena hacia el resto del mundo durante el periodo del estudio (septiembre 2025 a marzo 2026).
