

### 2. Base de datos inicial

## Plan de construcción de la base de datos - Wikipedia

La base se construirá mediante datos recolectados de Wikipedia, analizando archivos que muestran cuántas veces se buscó determinada especie en un periodo de seis meses. La página proporciona estos datos de forma pública en un archivo CVS específico para cada búsqueda que luego transformaremos en una base de datos que contenga las visitas a las especies que se encuentran catalogas en "peligro" y "peligro crítico".
